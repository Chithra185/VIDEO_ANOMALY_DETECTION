# VIDEO_ANOMALY_DETECTION

An AI-powered real-time video anomaly detection system developed using Deep Learning, Computer Vision, and Machine Learning techniques to identify suspicious or abnormal activities from surveillance footage.

## 📌 Features

* Real-time anomaly detection from video streams
* Detection of multiple abnormal activities such as:

  * Weapon Detection
  * Fire & Smoke Detection
  * Fighting Detection
  * Crowd Pushing Detection
  * Shoplifting Detection
  * Running Detection
  * Vandalism Detection
  * Large Crowd Detection
  * Accident Detection
* YOLO-based object detection models
* Flask-based web application interface
* Real-time prediction and visualization
* Graph generation and activity analysis

## 🚀 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Chithra185/VIDEO_ANOMALY_DETECTION.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd VIDEO_ANOMALY_DETECTION
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python app.py
```
